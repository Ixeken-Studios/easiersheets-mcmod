[Leer en Español](description_es.md)

### Skip the early-game kinetic setup for getting sheets and have them right away.

## ✨ How it works
Simply place any valid ingot onto a Depot, Sneak (Shift) + Right-Click with the Sturdy Hammer, and instantly smash it into a sheet!

![How it works](common/mod_easier.gif)

## 🌟 Key Features
- **100% Dynamic Compatibility:** The mod automatically reads Minecraft's internal recipe manager. If an ingot can be pressed by a Mechanical Press (Iron, Gold, Copper, Brass, or even ingots from other mods), the Sturdy Hammer can smash it!
- **Balanced Crafting:** Requires heavy materials to craft (Anvils, Iron Block, Lightning Rod, Copper), keeping it balanced for the early-to-mid game progression.
