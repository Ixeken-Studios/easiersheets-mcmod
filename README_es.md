# Create: Easier Sheets 🔨

Un addon ligero de NeoForge para el mod **Create**.

Para ver la descripción completa del mod y sus características, consulta [description_es.md](description_es.md).

[**Descargar en Modrinth**](https://modrinth.com/mod/create-easier-sheets)

## 📖 Descripción General
Introduce el **Sturdy Hammer** (Martillo Robusto), una herramienta que te permite prensar manualmente cualquier lingote válido para convertirlo en una placa (sheet) sobre un Depot sin necesidad de energía cinética.

## 🚀 Cómo ejecutar el proyecto

Este proyecto utiliza Gradle. Puedes ejecutar los siguientes comandos en el directorio raíz:

- **Ejecutar Cliente:** `./gradlew runClient`
- **Ejecutar Servidor:** `./gradlew runServer`
- **Compilar Mod:** `./gradlew build`

## 📂 Estructura del Proyecto
- `src/main/java/` - Lógica central, ítems e integraciones.
- `src/main/resources/` - Texturas, modelos, recetas y archivos de idioma.
- `common/` - Recursos de documentación.
